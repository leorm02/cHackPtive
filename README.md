# cHackPtive
Improved version of https://github.com/systematicat/hack-captive-portals

<h2>Dependencies</h2>

  On Arch: 
		sudo pacman -S nmap sipcalc

  On Ubuntu:
		sudo apt-get install sipcalc nmap
  
<h2>Usage</h2>
	
	git clone https://github.com/leorm02/cHackPtive.git
	cd cHackPtive
	sudo ./cHackPtive.sh
